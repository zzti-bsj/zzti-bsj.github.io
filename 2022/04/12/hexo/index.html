1. Hexo 简介

Hexo是一个简单、高效的博客框架。使用MarkDown解析文章，利用主题生成静态网页。

1.1 环境安装

这一步主要是在安装Hexo之前，安装其所依赖的Node.js和Git环境，在依次安装好所需的环境以及Hexo之后，就可以开始Hexo的开发了。

关于依赖环境的安装以及Start Hexo，请参考如下文档： Start Hexo

2. Hexo建站

在安装好Hexo之后，我们可以在终端键入hexo来简单了解一下Hexo的常用命令，在这里可以看到有初始化、帮助、版本的命令以及一些可选参数。

2.1 新建Hexo项目

使用hexo init命令来新建一个Hexo项目，这是我们一切开始的起点！

2.2 进入Hexo项目

在上一层目录下执行hexo只能够看到简单的一些可执行的命令，接下来进入到执行的init之后的目录，执行我们想做的操作。

在这个目录下，可以执行相关的命令来实现相应的操作，比如创建新文章、生成静态文件、部署网站等。

3. 部署到Github

3.1 创建Github仓库

新建一个Github repository，命名为<username>.github.io，这里的username是个人Github的账户昵称。

3.3 配置域名

  2. 在Github的<username>.github.io仓库找到 -> Setting -> Pages settings now has its own dedicated tab! Check it out here! -> Custom domain， 同样在这里写上域名，注意Source分支不要选择错误了
