链表：
链表是由一个又一个的节点组成的，链表的节点包括数据域和指针域，指针变量存储指向下一节点的地址。如下代码段所示。
typedef struct node
{
int num; //标记该字符是否是某一字符串的结尾
node *next[10];
}node;

phone list

Problem Description
Given a list of phone numbers, determine if it is consistent in the sense that no number is the prefix of another. Let’s say the phone catalogue listed these numbers:

Emergency 911
Alice 97 625 999
Bob 91 12 54 26
In this case, it’s not possible to call Bob, because the central would direct your call to the emergency line as soon as you had dialled the first three digits of Bob’s phone number. So this list would not be consistent.

Input

The first line of input gives a single integer, 1 ≤ t ≤ 40, the number of test cases. Each test case starts with n, the number of phone numbers, on a separate line, 1 ≤ n ≤ 10000. Then follows n lines with one unique phone number on each line. A phone number is a sequence of at most ten digits.

Output

For each test case, output “YES” if the list is consistent, or “NO” otherwise.

Sample Input
2
3
911
97625999
91125426
5
113
12340
123440
12345
98346

Sample Output
NO
YES
根据此题的需求，此题每个节点定义了10个next节点，但并不是每个节点都会用到。

/* 判断一组字符串中是否有一个字符串是另一个字符串的前缀
      注意有两种情况 */
#include<stdio.h>
#include<string.h>

typedef struct node
{
    int num;    //标记该字符是否是某一字符串的结尾
    node *next[10];
}node;

node memory[1000000];
int k;

int insert(char *s, node *T)
{
    int i, len, id, j;
    node *p, *q;//pq两个中间节点变量
    p = T;
    len = strlen(s);
    for (i = 0; i<len; ++i)
    {
        id = s[i] - '0';//字符对应的数字。
        if (p->num == 1)   //说明存在先前字符可以作为s的前缀----（先短后长）
            return 1;
        if (p->next[id] == NULL)//T的下一节点的地址不为空的话
        {   
            q = &memory[k++];
            q->num = 0;
            for (j = 0; j<10; ++j)
            q->next[j] = NULL;
            p->next[id] = q;
        }
        p = p->next[id];
    }
    for (i = 0; i<10; ++i)      //如果p的后继结点不为空的话说明s是先前字符的前缀----（先长后短）
    if (p->next[i] != NULL)
        return 1;
    p->num = 1;
    return 0;
}

int main()
{
    int m, n, flag, i;
    node *T;
    char s[15];
    scanf("%d", &m);
    while (m--)
    {
        k = 0;              //每次都从数组下标为0的地方开始分配内存，可以使内存循环利用，从而不会造成内存超限
        T = &memory[k++];//链表 中的 某个节点。
        T->num = 0;
        for (i = 0; i<10; ++i)
            T->next[i] = NULL;//后十个节点先赋NULL。
        flag = 0;
        scanf("%d", &n);
        while (n--)
        {
            scanf("%s", s);
            if (flag)
                continue;
            if (insert(s, T))
                flag = 1;
        }
        if (flag)
            printf("NO\n");
        else
            printf("YES\n");
    }
    return 0;
}

主代码分析：

1 每次进入insert,同时传入字符串s和T节点。
2 从字符串中提取整型数字，同时判断是否有完整的字符串已被标记（num是否为1）若是，return 1
3 以T节点为顶节点进行判断下一个节点是否为空，若空，创建q节点（&memory[k++]）并把q节点的十个next节点赋空，p->next[id]=q;。若不为空,执行第4部
4 p=p->next[id],对下一节点进行处理。
5 处理字符串完毕，for循环进行遍历，看s是否先前字符的前缀。
6 执行p->num=1;（一个完整字符串“存入”链表中的标记）
