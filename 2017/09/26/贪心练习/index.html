贪心练习（一）

贪心算法（又称贪婪算法）是指，在对问题求解时，总是做出在当前看来是最好的选择。也就是说，不从整体最优上加以考虑，他所做出的仅是在某种意义上的局部最优解。

经典的0-1背包问题：

我认为这种模板性的问题共两大类：
1 重量可以拆开的物品（用母函数易解决）
2 重量不可拆分的物品（以下方法是不错的选择）
这个问题是把物品整体装入剩余的空间，前提是在能够装入的情况下。

#include <iostream>
#include <string.h>
#define Max 20000   
using namespace std;
int main()
{
    int a[Max], i, j, w, v, M, N, temp;//w代表每个物品的重量，v代表每个物品的价值，
                                       //N代表背包的（总重量）容量，M代表有M个物品（或说是M种）
    memset(a, 0, sizeof(a));       //给数组a赋零（初始化），在G++和c++编译器
                                   //都能通过，不初始化只能在c++编译器通过
    cin >> M >> N;
    for (i = 1; i <= M; i++)
    {
        cin >> w >> v;           //每次输入一组物品的重量和价值,共M次。
        for (j = N; j >= w; j--) //每次j从最大背包容量（N）开始，到每个物品的w结束，
                                 //a[j-w]+v代表j-k空间内的物品价值+当前w空间内的物品价值。
        {
            temp = a[j - w] + v;
            if (temp>a[j])
                a[j] = temp;
        }
    }
    cout << a[N] << endl;
}

分析：

更新背包不同容量（从1-N）所能存入的价值量。更新每一次循环的a[N]

cin >> w >> v;

{
    temp = a[j - w] + v;
    if (temp>a[j])
        a[j] = temp;
}
