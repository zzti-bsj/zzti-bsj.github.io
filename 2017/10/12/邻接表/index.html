邻接表--如何用邻接表储存一个图

邻接表：

稀疏图用邻接表较为合适。
n,m分别代表顶点个数和边数。
接下来m行每行有三个整数x,y,z,表示x到y的边的权值为z。
1用数组建立邻接表

代码
#include<stdio.h>
int main()
{
    int n, m, i, k;
    int u[6], v[6], w[6];
    int first[5], next[5];
    scanf("%d%d", &n, &m);
    for (i = 1; i <= n; i++)
        first[i] = -1;
    //初始化first为-1，表示1-n都暂时没有边。
    for (i = 1; i <= m; i++)
    {
        scanf("%d%d%d", &u[i], &v[i], &w[i]);
        next[i] = first[u[i]];
        first[u[i]] = i;
    }
    for (i = 1; i <= n; i++)
    {
        k = first[i];
        while (k != -1)
        {
            printf("%d %d %d\n", u[k], v[k], w[k]);
            k = next[k];
        }
    }
    return 0;
}
