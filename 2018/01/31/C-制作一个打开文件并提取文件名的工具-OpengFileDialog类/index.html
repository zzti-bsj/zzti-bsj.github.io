C#:制作一个打开文件并提取文件名的工具--OpengFileDialog类

OpengFileDialog是类，并有openFileDialog控件，openFileDialog控件是OpengFileDialog的对象。
获取文件的文件名、文件扩展名、文件路径的信息实例中，用到了OpenFileDialog类及openFileDialog控件，此控件就是OpengFileDialog类的对象，主要还用到了string类型的Substring函数（用来分割子串），主要是用来分离文件的文件名、文件扩展名信息。
1 添加opengFileDialog控件（打开文件对话框控件）
2 添加button控件。
3 添加label控件。
主要的核心代码如下：

private void button1_Click(object sender, EventArgs e)
{
    if(openFileDialog1.ShowDialog()==DialogResult.OK)//判断是否选择了文件。
    {
        string all = openFileDialog1.FileName;//记录选择的文件的全路径。
        //获取文件路径。
        string path = all.Substring(0, all.LastIndexOf("\\") + 1);
        //获取文件名。
        string filename = all.Substring(all.LastIndexOf("\\") + 1, all.LastIndexOf(".") - (all.LastIndexOf("\\"))-1);
        //获取文件扩展名。
        string fileexc = all.Substring(all.LastIndexOf(".") + 1, all.Length - all.LastIndexOf(".") - 1);
        label1.Text = "文件路径：" + path;
        label2.Text = "文件名称：" + filename;
        label3.Text = "文件扩展名：" + fileexc;
    }

stirng 类型的Substring函数的第一个参数 是子串起始的索引，第二个参数是子串的长度。
Char类型的LastIndexOf(“\“)函数是用来获取某个字符串最后一个匹配位置的索引值。（返回值是索引值）
