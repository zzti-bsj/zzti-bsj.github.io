C#:做一个简单的时钟

通过制作一个简单时钟，我了解了一个类：DateTime，还有label、Timer控件。
不过只了解了DateTime类的一点皮毛，主要的还是学习C#，对于这些类，需要的时候再上网查吧。

DateTime是一个类。
获取年份、月份、小时、分钟、秒、的时间函数如下，这只是一些基本的功能函数。
DateTime.Now.Year.ToString(); 获取年份 // 2008
DateTime.Now.Month.ToString(); 获取月份 // 9
DateTime.Now.DayOfWeek.ToString(); 获取星期 // Thursday
DateTime.Now.DayOfYear.ToString(); 获取第几天 // 248
DateTime.Now.Hour.ToString(); 获取小时 // 20
DateTime.Now.Minute.ToString(); 获取分钟 // 31
DateTime.Now.Second.ToString(); 获取秒数 // 45
System.DateTime.Now.ToString(); 获取时间// 2018/1/26/14:20:15
除了以上功能函数外，还有其他形式函数，需要时上网查询。

制作时钟函数的大概过程：
（需要先添加控件，label、Timer控件）
1 控件函数全放到主类public partial class From1：From中。
2 label.Text=System.DateTime.Now.ToString();//放入Timer控件中，Timer的interval属性设置成100毫秒，每过100 毫秒就执行一次调用系统时间函数，并通过label控件显示在应用窗体中。
4 设置枚举的值：
enum MyDate
{
Sun = 0,
Mon = 1,
Tue = 2,
Wed = 3,
Thi = 4,
Fri = 5,
Sat = 6
}//用来设置星期几的值，以用switch来选择。
5 把以下要执行的操作放入From_Load中：（相当于C语言的主函数）
timer1.Start();//Timer控件执行，实现显示日期+时间的功能，用一个label控件显示。
int k = (int)DateTime.Now.DayOfWeek;//返回当前星期几的返回值k。

//switch通过获取的k的返回值来实现显示星期几的功能，用一个label控件显示。
        switch (k)
        {
            case (int)MyDate.Sun:
                label2.Text = "今天星期日";
                break;
            case (int)MyDate.Mon:
                label2.Text = "今天星期一";
                break;
            case (int)MyDate.Tue:
                label2.Text = "今天星期二";
                break;
            case (int)MyDate.Wed:
                label2.Text = "今天星期三";
                break;
            case (int)MyDate.Thi:
                label2.Text = "今天星期四";
                break;
            case (int)MyDate.Fri:
                label2.Text = "今天星期五";
                break;
            case (int)MyDate.Sat:
                label2.Text = "今天星期六";
                break;
        }
