杭电1010-搜索题简述-（DFS+剪枝技巧）

杭电1010：

  • DFS的基础知识题目
  • 可以用来理解并分析DFS类型的题目。
  • 做此类题首先把思路搞清楚，最好把路线图画出来写函数。

思路简述：（重点会有标记）

  • 简述dfs函数的写法。
  • 从S出发，输入是标记S的初始坐标，以作为函数dfs的形参。
  • dfs(int x,int y,int t)//t代表时间，每当走一步— t+1。
  • dfs(x,y,t)代表在坐标x,y位置时，已走的时间为t，这就是dfs函数参数的意义。
  • 则当x,y若为D时，此时写dfs函数的内容该这样写
if(map[x][y]=='D'&&t<T){
    b=true;//已找到解。
    return;
}
  • 那么dfs(x,y,t)该如何递归呢？
    • 下一步依据题目的要求，只能up down left right四个方向走，所以可以定义一个方向数组。
    • 四个方向是同样的操作所以考虑一个方向的操作，for循环解决。
    • 一个方向走的话有下一步的位置有三种情况：
      • 墙
      • 未标记的路
      • 已标记的路
    • 所以遇见墙和已标记的路就不执行下去了。只有未标记的路继续执行。
    • 每遍历一个位置要标记一个位置.
visited[xx][yy]=1;
dfs(xx,yy,t+1);
visited[xx][yy]=0;
    • 这样写是因为在当前dfs函数中xx,yy已经被访问过，所以visited[xx][yy]在dfs(xx,yy,t+1)之前执行了。
    • 剩下的为剪枝的操作技巧，在下面有笔记，所谓熟能生巧，往大神的路子上踏吧，dfs的基本思路是清晰了，但acm对自己的刁难还远远不仅如此，继续努力吧。

  • 剪枝:

    • 依据题目要求，当t=T时说明已经不用再往下进行了，所以return，
      这点很容易看出来。
    • 因为定义了一个全局变量flag，当flag为真时，结果已经出来了，不用继续往下遍历，所以当flag为false时，for循环继续执行。
    • 很容易想到，在主函数中，当且仅当T时刻门打开，所以判断T与识末位
      置的奇偶性是否相同，不相同也没有必要往下执行了。

  • 细节：

    • 因为是多组输入数据，所以注意变量，每次执行某一组数据时，是否临时的变量更新了而不是上一组数据残留的赋值。

  • 新学到的东西:

    • C的输入格式，scanf(“%c”,&ch);这个输入格式会把空行（\n）保留在ch中，当需要用输入一组字符的话—-伪码：
      `
char ch[5];
for(int i=0;i<3;i++)
{
    ch[i]=getchar();
}
getchar();//这个是缓存空行（\n）的。

`

  • 附上杭电1010源码：
#include <iostream>
#include<stdio.h>
#include<algorithm>
#include<string.h>
using namespace std;
char map[8][8];
int visited[8][8];
int dir[4][2]={1,0,-1,0,0,1,0,-1};
int sx,sy;
int N,M,T;
bool flag;
int ds;
void dfs(int x,int y,int t)
{
    if(map[x][y]=='D'&&t==T)
        {
        flag=true;
        return ;
        }
    if(t==T)
        return;
    if(flag==false)
    for(int i=0;i<4;i++)
    {
        int xx=x+dir[i][0];
        int yy=y+dir[i][1];
        if(map[xx][yy]=='X'||visited[xx][yy]||xx>=N    ||yy>=M||xx<0||yy<0)
        continue;
        visited[xx][yy]=1;
        dfs(xx,yy,t+1);
        visited[xx][yy]=0;
    }
    return;
}
int main()
{
    while(cin>>N>>M>>T&&N&&M&&T)
    {
        memset(visited,0,sizeof(visited));
        getchar();
        flag=false;
        for(int i=0;i<N;i++)
        {
            for(int j=0;j<M;j++)
            {
                map[i][j]=getchar();
                //cin>>map[i][j];
                if(map[i][j]=='S')
                {
                    sx=i;
                    sy=j;
                }
            if(map[i][j]=='D')
                ds=i+j;
            }
        getchar();
        }
        visited[sx][sy]=1;
        if (((ds - sx - sy) & 1) == (T & 1))
            dfs(sx, sy, 0);
        if(flag)
            printf("YES\n");
        else
            printf("NO\n");
    }
    return 0;
}
