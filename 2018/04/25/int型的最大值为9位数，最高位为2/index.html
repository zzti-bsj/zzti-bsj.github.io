int型的最大值为10位数，最高位为2

碰到了代码中定义最大值的问题，感觉不太熟练，于是上网查资料解决了一下。

  • #define inf 0xffffffff//宏定义，不规定类型，一般使用宏定义定义最大值。一般不爆内存。

  • const int inf=0xffffffff;//不可变的值。

  • 用十六进制表示int型的最大数是多少？

1、 正好是0Xffffffff(4294967296)表示为int无符号的最大值。所以一般定义最大值用宏定义为0xffffffff

2、 int 32位有符号型最大值为九位数，十位上是2。

- 有符号 无符号
int -2^31 ~ 2^31-1(-2147483648~2147483647) 0~2^32-1(4294967295)
  • 一秒大概对应多大的运算量？
    如下：一般情况下是10^8的计算量。
差 中 强
10^8 4*10^8 10^9
