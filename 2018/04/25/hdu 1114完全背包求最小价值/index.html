hdu 1114完全背包求最小价值
轻松ac，简直完全背包的模板题，不过是要求 ++背包装满++ 且求最小价值的。

完全背包的题型如下：

  • 求最小价值：要求恰好装满背包，那么在初始化时除了dp[0]为0其它dp[1..V]均设为∞
    • dp[j]=min(dp[j],dp[j-w[i]]+v[i]);
  • 求最大价值：要求恰好装满背包，那么在初始化时除了dp[0]为0其它dp[1..V]均设为-∞
    • dp[j]=max(dp[j],dp[j-w[i]]+v[i])
      (其实求最大的时候不用赋值dp[i]为负无穷。)

++附上本人原创代码，++

代码
//听说求最小需要赋值的时候把dp[0]=0;其他赋正无穷。
#include<iostream>
#include<string.h>
using namespace std;
int inf=0x3f3f3f;
int T;
int n,E,F;
int v[505],w[505];
int dp[10010];
int main()
{
    cin>>T;
    while(T--)
    {
        cin>>E>>F;
        int ans=F-E;
        cin>>n;
        for(int i=0;i<n;i++)
        {
            cin>>v[i]>>w[i];
        }
        memset(dp,inf,sizeof(dp));
        dp[0]=0;
        for(int i=0;i<n;i++)
            for(int j=w[i];j<=ans;j++)
            dp[j]=min(dp[j],dp[j-w[i]]+v[i]);
        if(dp[ans]<inf)
            cout<<"The minimum amount of money in the piggy-bank is "<<dp[ans]<<'.'<<endl;
        else
            cout<<"This is impossible."<<endl;
    }
    return 0;
}

狠狠的刷背包！！
