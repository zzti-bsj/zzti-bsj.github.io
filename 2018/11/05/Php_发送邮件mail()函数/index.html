Php_发送邮件mail()函数

  • 2018-11-05
  • Bsj

环境配置

  • 配置php.ini文件
  • 配置sendmail.ini文件（需要下载sendmail）
  • qq邮箱的授权码设置

php.ini配置

主要配置三点：


SMTP = smtp.qq.com
smto_port = 25
//(邮箱服务端口)

sendmail_from = yourname@qq.com
//自己的邮箱账号

sendmail_path = "D:\sendmail\sendmail.exe -t"
//自己安装sendmail的路径

sendmail下载及配置

sendmail下载之后，解压到自定义目录，如我的是：D:\sendmail

  1. 配置sendmail.ini
smtp_server=smtp.qq.com
smtp_port=25

//开启 log 方便排错，生成的log文件在sendmail根目录
error_logfile=error.log
debug_logfile=debug.log

auth_username=你的邮箱地址
auth_password=你的授权码

force_sender=你的邮箱地址 （和auth_username一样）
  1. qq授权码的获取

登录qq邮箱mail.qq.com，
选择“设置”–“账号”–“POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务”

开启如下服务,生成授权码，并填写在sendmail中授权码的位置。

配置完成！

测试代码

    <?php
    $to = '*********@qq.com';
    //填写接受者的邮箱

    $subject = 'Test mail';
    $message = '恭喜！（测试邮箱发送！）';
    mail($to, $subject, $message);
    echo "Mail Sent.";
    ?>
