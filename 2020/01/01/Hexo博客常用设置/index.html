Hexo博客常用设置

详情语法请参考网址http://www.appinn.com/markdown/#codehttps://segmentfault.com/a/1190000010223222：

创建文章：

  1. 在hexo文件夹下， hexo new “文章标题”
  2. 写标题 title
  3. 写日期 date
  4. 写标签 tags
  5. 添加文章内容

发布文章： hexo d -g 如果发布时出现错误，执行 npm install hexo-deployer-git –save

图片的插入： 如

（路径就按照这个路径格式，因为文章在与imag路径的 _post 下。）

主题的更换：

  1. 在themes文件夹下，把主题下载下来，执行git clone [主题地址]
  2. 打开Hexo文件夹下的_config.yml文件，修改里面的theme为主题文件夹名称
  3. 使用命令hexo server测试，访问http://0.0.0.0:4000/会看到主题已经更改成功
  4. 附：下载主题的网址https://hexo.io/themes/

当前使用的主题教程：Anisina主题教程

博客图片设置：

favicon: imag/ico.ico      
sidebar-avatar: imag/favicon.png
post-default-img: /imag/p.jpg #博文默认的图片 绝对路径
post-default-img-mobile: /imag/p.jpg #移动端显示的图片 这个可以是相对路径

//imag前面加/代表绝对路径表示从根目录开始找，然后不加/代表相对路径，
从当前路径开始找，比如imag/p.jpg代表在imag当前的文件夹下找p.jpg这张图片。

测试标题的背景图像 header-img.

在标题位置加上 header-img: "/imag/ppp.jpg" 即可以设置自己喜欢的图片为header！
header-img: "/imag/header/ppp.jpg"
