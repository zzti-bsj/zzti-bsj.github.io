з”ҹжҲҗеҷЁзҡ„next()е’Ңsend()ж–№жі•

next(c) : ејҖеҗҜз”ҹжҲҗеҷЁпјҢдјҡиҝ”еӣһдёҖдёӘеҖјпјҢз„¶еҗҺеҶҚyieldеӨ„зӯүеҫ…жҺҘж”¶дј жқҘзҡ„еҸӮж•°

еҗҜеҠЁз”ҹжҲҗеҷЁд№ӢеҗҺпјҡ

result = c.send(value) -> з”ҹжҲҗеҷЁе…ҲжҺҘеҸ—дёҖдёӘеҖјпјҢеҶҚиҝ”еӣһдёҖдёӘеҖј

def test():
    for i in range(20):
        c = yield i
        print("еҖјпјҡ", c)


if __name__ == '__main__':
    c = test()
    # 1.ејҖеҗҜз”ҹжҲҗеҷЁ
    print("first value:", c.__next__())

    # 2.дҪҝз”Ёsendжү§иЎҢз”ҹжҲҗеҷЁ
    result = c.send(31)
    print(result)

    result = c.send(31)
    print(result)

    result = c.send(31)
    print(result)

yieldзҡ„дҪңз”ЁжңәеҲ¶

  yieldжү§иЎҢе®ҢеҪ“еүҚеҮҪж•°еҗҺпјҢжҡӮеҒңжү§иЎҢе№¶иҝ”еӣһдёҖдёӘеҖјгҖӮ

дёҖж®өжҳ“жҮӮзҡ„д»Јз Ғ

д»Јз Ғ

def foo():
    print("starting...")
    while True:
        res = yield 4
        print("res:",res)
g = foo()
print(next(g))
print("*"*20)
print(g.send(7))
