jQuery概述（一）

什么是jQuery?

  jQuery是js的一个高效的库

js的特性

  1. 特性：write less, do more
  2. 作用：
    • 选取、操作HTML、操作HTML的事件
    • 操作css
    • 制作js特效
    • ajax功能
    • utilities

      jQuery的引用

      1.官网下载jQuery

      1、官网下载（.min.js（精简，用于发布时使用）/.js（用于开发时使用，方便debug））：www.jQuery.com
      2、导入代码(src中填写自己的jQuery文件路径)：
      <script type="text/javascript" src="js/jquery-3.4.1.js"></script>

2.从谷歌或者微软的CDN导入jQuery

这种方式会更快一点，因为许多用户已经加载过谷歌或者微软的jQuery，从本地缓存中获取，会更快。
1、从微软导入：
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.0/jquery.min.js"> </script>
2、从谷歌导入：
<script src="http://ajax.aspnetcdn.com/ajax/jQuery/jquery-1.8.0.js"> </script>

jQuery的语法

jQuery语法

$(document).ready(func..)

  1. $：代表jQuery
  2. ()：选择器
  3. action()：动作

jQuery命名冲突

有时候在jquery的函数中也可能使用到$，造成命名冲突。需要给jQuery重命名解决

文档就绪函数

  $(document).ready(func..)
  所有的jQuery代码放在文档就绪函数中，这样做是为了防止文档未加载完成之前运行jQuery代码。
比如：

  1. 试图隐藏一个不存在的元素
  2. 获取未加载完成的图像的大小

    jQuery规范

  3. 把所有 jQuery 代码置于事件处理函数中
  4. 把所有事件处理函数置于文档就绪事件处理器中
  5. 把 jQuery 代码置于单独的 .js 文件中
  6. 如果存在名称冲突，则重命名 jQuery 库

jQuery选择器

  1. 元素选择器
    $("p") 选取所有的p标签
  2. 属性选择器
    选取所有属性带有$("[href]")的标签
  3. css选择器
    $("p").css("background-color","red");

更多更选择器样式参考www.w3school.com.cn的jQuery教程

  1. 事件处理程序，是HTML页面中发生某些事件的时，所调用的方法
  2. jQuery事件处理程序是jQuery的核心函数
