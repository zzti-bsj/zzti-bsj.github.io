几种常见的加密方式

md5（Message Digest Algorithm）

  md5，中文名称为消息摘要算法，字面意思上来说，是一种采用摘要进行加密的算法。
  md5加密的特点：
  1.加密之后是一个32位的字符串
  2.属于不可逆加密方式
注：md5算法虽然不可逆，但是对于一些简单的字符串，还是可以通过md5加密之后暴力匹配出来的。
（网上有一些专门破解md5加密的网站，有兴趣的可以搜索一下，下面给一段代码，可以生成简单的md5加密之后的数据，贴在解密网站上是可以解密的。）
  代码示例：


# 其实在加密之前给string编码一下，就可以避免汉字问题。
# 注：utf-8使用三个字符表示汉字，gbk使用两个字符表示汉字，给汉字不同的编码，使用md5之后加密的字符串也不同。
# encode()默认编码成urf-8格式

# 导入hashlib 包
from hashlib import md5
new_md_object = md5()  # 获取md5对象
new_md_object.update(string.encode())  # update()将字符串载入md5对象，获得md5加密
new_digest_string = new_md_object.hexdigest()  # hexdigest()返回一个md5加密之后的32位字符串

AES（Advanced Encryption Standard）

  AES：高级加密标准，DES是它的前任标准。

DES（Data Encryption Standard）

  DES：数据加密标准

哈希+盐

  哈希+盐，也是一种常用的加密手段。哈希，是利用加密方法，盐：一般是随机字符串。通过哈希+盐的方式进行加密，使加密的安全性更高。
  比如说，md5可以通过暴力碰撞的方式进行破解，那密码的安全性就受到了威胁。在这种情况下，我们把原字符串+盐，然厚进行加密，就得到了一个新的md5值，这样的话，提升了加密的安全性。

# 简单举例，实际上这个盐并不一定这么简单：

from hashlib import md5
s = '这是随机字符串'
new_md5 = md5()
new_string = new_md5.update((string+s).encode).hexdigest()
