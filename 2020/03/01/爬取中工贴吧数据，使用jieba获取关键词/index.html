css选择器三种查找的形式：

  1. .class # 类选择
  2. #id # id选择
  3. [target=”？？”] # 属性选择
  4. ::text # :: 取内容
  5. ::attr(attribute) # 取属性的内容

css选择器实际使用时的问题：

  1. 怎么选择相对路径下的标签？

结巴分词的浅记

代码不长，贴下：

# -*- coding:utf-8 -*-
# 作者：bsj
# 邮箱：2779090360@qq.com
# 时间：2019/7/24
import jieba
from collections import Counter


def output_top_n():
    list = []
    with open('zut_tieba.csv') as f:
        for line in f:
            # 对每一行使用jieba处理
            cut = jieba.cut(line)
            # 和空列表合并
            list += cut
    couter = Counter(list).most_common(10000)
    with open("key_word.txt", 'a+') as f:
        for cou in couter:
            f.write(str(cou))
            f.write("\n")
            print(cou)
        f.close()


output_top_n()

  使用jieba模块调用cut函数，传入字符串参数，返回一个关键字组成的生成器，这里我使用了列表合并，把关键字写入同了一个列表中。

精准匹配

str = "我是一个爱学习的Coder，学习也爱我，我没有女朋友"
gen = jieba.cut(str)
print(",".join(gen))

>>>我,是,一个,爱,学习,的,Coder,，,学习,也,爱,我,，,我,没有,女朋友

全模式

全模式解析的好像更全了。

import jieba
str = "我是一个爱学习的Coder，学习也爱我，我没有女朋友"
gen = jieba.cut(str, cut_all=True)
print(",".join(gen))

附上代码（包含scrapy/jieba模块，在scrapy项目下可以运行此代码）： 

# -*- coding:utf-8 -*-
# 作者：bsj
# 邮箱：2779090360@qq.com
# 时间：2019/7/24
"""
css选择器的使用方法：
1、含有多个类的标签，类之间使用.连接
2、空格就代表了相对路径和绝对路径
3、选择某个属性为***的标签： response.css('a[href="***"]')
4、使用单引号

传参：
1、scrapy.Request()使用meta传递参数。
2、在背传入的函数中，使用meta[key]获取接受的参数。
"""
import scrapy


class ZutTb(scrapy.Spider):
    name = "spider_zut_tb"

    # start_urls = [
    #     "http://tieba.baidu.com/f?kw=%D6%D0%D4%AD%B9%A4%D1%A7%D4%BA&fr=ala0&tpl=5"
    # ]

    # 定义一个写入数据的函数
    # 存入到csv表中
    def save_to_csv(self, pre_data):
        with open('zut_tieba.csv','a+') as f:
            f.write(pre_data + "\n")
            f.close()

    # 这里面，每次使用scrapy.Request()请求一个网页
    def start_requests(self):
        urls = [
            "http://tieba.baidu.com/f?kw=%D6%D0%D4%AD%B9%A4%D1%A7%D4%BA&fr=ala0&tpl=5"
        ]
        for url in urls:
            yield scrapy.Request(
                url=url,
                callback=self.parse,
            )

    def parse(self, response):
        # 使用css解析网页
        # 使用response解析
        # 1、一个标签多个类之间使用.连接
        # 2、使用空格即有相对路径的作用

        # 获取某个贴
        all_tiebas = response.css("li.j_thread_list.clearfix")
        for tieba in all_tiebas:
            # 或取某个贴吧的回复数
            num_res = tieba.css("span.threadlist_rep_num.center_text::text").extract_first()

            # 获取贴吧主题
            tieba_title = tieba.css('a[rel="noreferrer"]::text').extract_first().strip()

            # 获取贴吧的href
            # 3、选择某个属性为**的标签
            tieba_href = "http://tieba.baidu.com" + tieba.css('a[rel="noreferrer"]::attr(href)').extract_first()
            # print(tieba_href)

            # 把数据传入到另一个页面？
            if int(num_res) >= 5:
                try:
                    yield scrapy.Request(
                        url=tieba_href,
                        callback=self.get_pre_page_info,
                        meta={
                            'tieba_title': tieba_title,
                        }
                    )
                except Exception as e:
                    print("异常")
                    print(e)

        # 下一页
        next_page = response.css('a.next.pagination-item::attr(href)').extract_first()
        if next_page is not None:
            next_page = response.urljoin(next_page)
            print(next_page)
            yield scrapy.Request(
                url=next_page,
                callback=self.parse,
            )


    # 获取每一页的信息
    def get_pre_page_info(self, response):
        # 贴吧 吧主
        all_content_com = response.css('div.d_post_content_main')
        for content_com in all_content_com:
            # 获取content和com
            content = content_com.css('div.d_post_content.j_d_post_content.clearfix::text').extract_first().strip()
            # 我获取不到这个数据
            # com = content_com.css('div').extract()

            # 已经获取到字符串
            self.save_to_csv(content)
            print(content)


# 使用代码段运行scrapy爬虫
if __name__ == '__main__':
    from scrapy import cmdline
    cmdline.execute('scrapy crawl spider_zut_tb'.split(" "))
