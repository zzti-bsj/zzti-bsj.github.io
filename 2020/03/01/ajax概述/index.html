ajax概述

  • 2020-03-01
  • Bsj

ajax

1. ajax是什么？

  1、ajax是一种用于创建快速动态页面的艺术
  2、ajax在不重新加载页面的前提下，与服务器交换数据更新当前页面。

2. ajax创建对象的两种方式

1、创建XMLHttpRequest对象

var xhrhttp = new XMLHttpRequest();

2、创建ActiveX对象

var xhrhttp = new ActiveXObject("Microsoft.XMLHTTP);

3. ajax发送请求

open()和send()方法

  ajax发送请求的两个函数分别是open()和send()
  两种发送请求的方式：get、post，其中send()可以用来传递post的参数。
参数：

  1. open()参数
    param1：method，请求方法get/post
    param2：url，请求的网址
    param3：async，是否异步执行，默认为true
  2. send()参数
    param1：strings仅用于post请求

代码段示例：

4. onreadystatechange

  1. 三个属性
    • onreadystatechange：当readystate改变时调用这个函数
    • readystate：存有xmlhttprequest的状态
      0：请求初始化
      1：服务器连接已建立
      2：服务器已接收请求
      3：服务器已响应请求
      4：请求已完成，且响应已就绪
    • status：响应状态
      200：成功
      404：页面未找到
  2. Callback函数：把函数作为参数传入的函数称为callback函数。

    ajax能做的事

  3. 动态判断用户的输入，返回相应的数据
  4. 与数据库实时交互，返回用户查询的信息
  5. 读取xml文件
