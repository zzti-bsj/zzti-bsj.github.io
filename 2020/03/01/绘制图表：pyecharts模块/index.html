绘制图表：pyecharts模块

  • 2020-03-01
  • Bsj

pyechars模块

pip install pyecharts==0.5.11
#pyecharts=echarts+python
  pycharts模块用来生成ecahrts的图表库。
  优点

  1. 动态美观
  2. 具有交互性，方便数据的对比分析
  3. 生成动态的echarts网页，也可以在flask/django中集成使用

    pyecharts制图实例

    pyecharts模块包含的图表： 
    Bar（柱状图/条形图） 
　　Bar3D（3D 柱状图） 
　　Boxplot（箱形图） 
　　EffectScatter（带有涟漪特效动画的散点图） 
　　Funnel（漏斗图） 
　　Gauge（仪表盘） 
　　Geo（地理坐标系） 
　　Graph（关系图） 
　　HeatMap（热力图） 
　　Kline（K线图） 
　　Line（折线/面积图） 
　　Line3D（3D 折线图） 
　　Liquid（水球图） 
　　Map（地图） 
　　Parallel（平行坐标系） 
　　Pie（饼图） 
　　Polar（极坐标系） 
　　Radar（雷达图） 
　　Sankey（桑基图） 
　　Scatter（散点图） 
　　Scatter3D（3D 散点图） 
　　ThemeRiver（主题河流图） 
　　WordCloud（词云图）

　　用户自定义

　　Grid 类：并行显示多张图 
　　Overlap 类：结合不同类型图表叠加画在同张图上 
　　Page 类：同一网页按顺序展示多图 
　　Timeline 类：提供时间线轮播多张图

以下是几种常用的图表实例代码
  柱状图/柱状3D图/折线图/折线3D图/饼图/地图/散点图/3D散点图/雷达图/k线图

主要列举了以下几种常用的图的实例示例

from pyecharts import Bar, Bar3D, Pie, Line, Line3D, Map, Scatter, Scatter3D, Radar, Kline, WordCloud

柱状图

# 柱状图---------------------------
"""
add(name, x_axis, y_axis)
x_axis 横坐标轴数据
y_axis 纵坐标轴数据
x_axis/y_axis的数据个数需要一致
"""
bar = Bar('柱状图', '测试')
x_axis = ['one', 'two', 'three', 'four']
bar.add('b', x_axis, [1, 2, 3, 5])
bar.add('a', x_axis, [1, 2, 3, 4])
bar.render()

柱状3D图

# Bar3D图---------------------------
"""
add(name, x_axis, y_axis, data)
其中：data是x，y的数据集，每一行代表一条数据(x,y,data)，每一列代表一个维度
设置网格透明度（默认值为1）：grid_opacity=1
设置网格颜色/样式（没测出来什么意思）：grid_shading=1
"""

bar_3d = Bar3D()
x_axis = ['one', 'two', 'three']
y_asis = ['python', 'java', 'php', 'c++']
data = [
        [0, 0, 5], [0, 1, 1], [0, 2, 3], [0, 3, 2], [1, 0, 7], [1, 1, 15],
        [1, 2, 6], [1, 3, 20], [2, 0, 11], [2, 1, 9], [2, 2, 0], [2, 3, 2], ]
bar_3d.add('a', x_axis, y_asis, data, grid3d_opacity=0.5, grid3d_shading='realistic')
bar_3d.render()

折线图

# 折线图----------------------------
"""
add(name, x_axis, y_axis)

设置是否是阶梯型 bool ：is_step
设置是否平滑 bool：is_smooth

"""
line = Line('折线图', '子标题')
x_axis = ['one', 'two', 'three', 'four']
y_axis = [100, 50, 75, 82]
line.add('图表名称', x_axis, y_axis, is_smooth=True, is_step=False, is_fill=False)
line.render()

折线图3D

# 折线图3D---------------------------
"""
add(name, data)
name: 图表名称
data: 每一行是一行数据，每一列是一个维度

"""
line_3d = Line3D('Line3D折线图', '这是子标题')
data = [
    [1.5, 0, 7.3], [2, 0, 7.8], [2.5, 0, 8], [3, 0, 8], [3.85, 0, 7.85], [4.4, 0, 7.5], [4.6, 0, 7.5], [5.15, 0, 7.85], [6, 0, 8], [6.5, 0, 8], [7, 0, 7.8], [7.5, 0, 7.3], [4.5, 0, 7.5], [4.5, 0.6, 8], [4.5, 1, 8.5], [4.5, 1.5, 8.8]
]
line_3d.add('图表名称', data, grid3d_opacity=0.5, is_smooth=True)
line_3d.render()

饼状图

# 饼状图-----------------------------
"""
add(name, attr, value)
每一个attr对应一个比例值（整数）
"""
pie = Pie()
attr = ['python', 'java', 'php']
value = [15, 15, 15]
pie.add('饼状图', attr, value, rosetype='area', radius=[10, 75], center=[50, 60])
pie.render()

地图map

# 地图 map-------------------------
"""
参数：
name: 地图名称
attr: 标记城市
value：标记值
maptype（显示地图区域）：国家/省份/城市

"""
map = Map()
attr = ['项城市', '沈丘县', '鹿邑县', '太康县']
values = [1, 1, 1, 4]
map.add('china', attr, values, maptype='周口', is_label_show=True)
map.render()

Scatter 直角坐标系散点图

# Scatter 直角坐标系散点图-----------------
"""
参数：
name: 图表名称
x_axis: x轴参数
y_axis: y轴参数
x_axis/y_axis的list值都必须为数值


"""
scatter = Scatter()
x_axis = [1, 2, 3, 4]
y_axis = [1, 2, 3, 4]
extra_data = [1, 2, 3, 4]
scatter.add('scatter', x_axis, y_axis, extra_data=extra_data)
scatter.render()

3D Scatter 直角坐标系散点图

# Scatter3D 散点图3D------------------
"""
参数：
name: 图表名称
data: 每一行是一行数据，每一列是一个维度
"""
scatter_3d = Scatter3D()
data = [
    [1.5, 0, 7.3], [2, 0, 7.8], [2.5, 0, 8], [3, 0, 8], [3.85, 0, 7.85], [4.4, 0, 7.5], [4.6, 0, 7.5], [5.15, 0, 7.85], [6, 0, 8], [6.5, 0, 8], [7, 0, 7.8], [7.5, 0, 7.3], [4.5, 0, 7.5], [4.5, 0.6, 8]
]
scatter_3d.add('Scatter3D', data=data)
scatter_3d.render()

k线图

# Kline k线图--------------------
kline = Kline()
x_axis = ['one', 'two', 'three']
y_axis = [
    [1, 2, 1, 5],
    [2, 4, 1, 15],
]
kline.add('k线图', x_axis, y_axis)
kline.render()

词云图

# WordCloud 词云图--------------------
"""
shape:设置词云图的展示格式
"""
word_cloud = WordCloud()
attr = ['one', 'two', 'three', 'four']
value = [1, 2, 3, 4]
word_cloud.add('WordCloud', attr, value, shape='star')
word_cloud.render()

组合图

overlap组合图-----------------------
import pyecharts as pye
attr = ['A', 'B', 'C', 'D', 'E', 'F']
v1 = [10, 20, 30, 40, 50, 60]
v2 = [38, 28, 58, 48, 78, 68]
bar = pye.Bar("Line - Bar 示例")
bar.add("bar", attr, v1)
line = pye.Line()
line.add("line", attr, v2)
overlap = pye.Overlap()
overlap.add(bar)
overlap.add(line)
overlap.render()
