等价多米诺骨牌对的数量

题目描述：
  给你一个由一些多米诺骨牌组成的列表 dominoes。
如果其中某一张多米诺骨牌可以通过旋转 0 度或 180 度得到另一张多米诺骨牌，我们就认为这两张牌是等价的。
形式上，dominoes[i] = [a, b] 和 dominoes[j] = [c, d] 等价的前提是 a==c 且 b==d，或是 a==d 且 b==c。
  在 0 <= i < j < dominoes.length 的前提下，找出满足 dominoes[i] 和 dominoes[j] 等价的骨牌对 (i, j) 的数量。

输入：dominoes = [[1,2],[2,1],[3,4],[5,6]]
输出：1

思路：
1、双重循环，暴力。不用想，超时。
2、暴力+剪枝，超时。把列表中的元素一个一个的pop()出来，按题目描述的两种形式，如果存在于剩下的列表元素中，则，num+1。
1.剩下的列表中所有元素是否唯一？是否会被忽略
2.剩下的列表中的元素是否是[x,x]形式？是否会重复
3、哈希表。
  从列表中取出一个元素pop()，判断题目描述的两种形式是否在于哈希表中，如果不存在，对应的元素在哈希表中的值设置为1，如果存在，则对应的元素在哈希表中的值+1，最后遍历整个哈希表，取其值做组合运算。

代码： 

def numEquivDominoPairs(self, dominoes):
        """
        :type dominoes: List[List[int]]
        :rtype: int
        """
        dic = {}
        while dominoes:
            # 取出最后一个
            dom = dominoes.pop()
            # 如果剩下的元素中有与当前元素重复的呢？
            # 如果dom[0], dom[1]相同呢？如果两者不同呢？
            # 在里面，有几个？
            dom = [str(i) for i in dom]
            if "".join([dom[0], dom[1]]) in dic:
                # 字典的键有什么要求?
                dic["".join([dom[0], dom[1]])] += 1
            elif "".join([dom[1], dom[0]]) in dic:
                dic["".join([dom[1], dom[0]])] += 1
            else:
                dic["".join([dom[0], dom[1]])] = 1
        # 遍历一遍字典
        length = 0
        for d in dic.values():
            if d != 1:
                length += d * (d - 1) / 2
        # 计算组合数
        return length
