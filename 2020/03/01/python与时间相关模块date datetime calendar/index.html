  • 2020-03-01
  • Bsj

三个模块time/datetime/calendar

常用的功能

对应我们编程，经常使用的功能：

  1. 获取当前的年月日-时分秒、年月日、时分秒。
  2. 获取时间戳
  3. 延时
  4. 程序处理时间
  5. 今天是周几？
  6. 获取年、月、日、时、分、秒

经常使用的三个模块time、datetime、calendar

  1. time模块
    • time() #获取时间戳
    • sleep() # 设置延时 参数单位为秒
  2. datetime（年月日-时分秒模块）
    • datetime类
      • 无参构造实例（默认当前年月日-时分秒）/有参构造实例(年月日-时分秒)
      • fields和methods
      • 常用方法：today()/date()/time()/weekday()
    • date类
      • 无参构造实例（默认当前年月日）/有参构造实例(年月日)
      • fields和methods
      • 常用方法：today()/weekday()
    • time类
      • 无参构造实例（默认当前时分秒）/有参构造实例(时分秒)
      • fields和methods
      • 个人认为没有什么常用方法，但可以通过调用函数之后返回的对象调用year/month/day等属性，以上的类 （同）。
  3. calendar（日历模块）
    • 获取日历相关的日期，一般我们日历都是date（year,month,day,weekday）
    • itermonthdates()/itermonthdays()/itermonthdays1…看起来都类似。
