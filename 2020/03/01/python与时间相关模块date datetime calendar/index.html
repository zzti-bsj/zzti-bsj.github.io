python与时间相关模块date/datetime/calendar

  • 2020-03-01
  • Bsj

三个模块time/datetime/calendar

常用的功能

对应我们编程，经常使用的功能：

  1. 获取当前的年月日-时分秒、年月日、时分秒。
  2. 获取时间戳
  3. 延时
  4. 程序处理时间
  5. 今天是周几？
  6. 获取年、月、日、时、分、秒

经常使用的三个模块time、datetime、calendar

  1. time模块
    • time() #获取时间戳
    • sleep() # 设置延时 参数单位为秒
  2. datetime（年月日-时分秒模块）
    • datetime类
      • 无参构造实例（默认当前年月日-时分秒）/有参构造实例(年月日-时分秒)
      • fields和methods
      • 常用方法：today()/date()/time()/weekday()
    • date类
      • 无参构造实例（默认当前年月日）/有参构造实例(年月日)
      • fields和methods
      • 常用方法：today()/weekday()
    • time类
      • 无参构造实例（默认当前时分秒）/有参构造实例(时分秒)
      • fields和methods
      • 个人认为没有什么常用方法，但可以通过调用函数之后返回的对象调用year/month/day等属性，以上的类 （同）。
  3. calendar（日历模块）
    • 获取日历相关的日期，一般我们日历都是date（year,month,day,weekday）
    • itermonthdates()/itermonthdays()/itermonthdays1…看起来都类似。

测试源码

import time, datetime, calendar

"""
# 时间戳：从1970年1月1日到现在的所有秒数之和

"""
# time模块常用方法

# 1、time() 返回时间戳
timestamp = time.time()

# 2、sleep() 延迟函数，参数是秒数，可以为float型
# time.sleep(2)

# 3、process_time()/process_time_ns() 程序处理时间  程序处理时间的纳秒
processtime = time.process_time()
processtime_ns = time.process_time_ns()
print(processtime)
print(processtime_ns)

# 4、ctime()/asctime() 返回当前时间的字符串形式
cur_time_str = time.asctime()
print(cur_time_str)
cur_time_str = time.ctime()
print(cur_time_str)

# 5、localtime() 返回一个元组表达式形式的列表
cur_time = time.localtime()
print(cur_time)

# datetime模块
"""
datetime模块中的三个类class
    1、datetime：年月日-时分秒
        （1）既然是类，那么就有fields和methods，调用datetime创建一个实例，需要传入时间（整型:年、月、日、时、分、秒。。。）
        （2）可以根据创建的实例获取其属性值
        （3）不传入参数直接调用提供的方法，默认参数是当前的年月日-时分秒。methods: today()/date()/time()/weekday()/other methods
    2、date：年月日
        （1）既然是类，那么就有fields和methods，调用date创建一个实例，需要传入时间（整型:年、月、日）
        （2）可以根据创建的实例获取其属性值
        （3）不传入参数直接调用提供的方法，默认参数是当前的年月日。methods: today()/weekday()/other methods
    3、time：时分秒
        （1）既然是类，那么就有fields和methods，调用time创建一个实例，需要传入时间（整型:时、分、秒）
        （2）可以根据创建的实例获取其属性值
        （3）不传入参数直接调用提供的方法，默认参数是当前的时分秒。

***！之前一直对时间很模糊，现在基本了解了程序中具体到分秒的时间：date（年月日） - time（时分秒），同时这对应着datetime类的两个方法date()/time()，返回什么值自然一想就知道了，肯定是date()返回年月日，time()返回时分秒
"""
# 
# 返回时分秒
x = datetime.datetime(2019, 7, 26, 13, 24, 55).time()
print(x)

# 返回ctime类型的字符串
x = datetime.datetime(2019, 7, 26, 13, 24, 55).ctime()
print(x)

# 返回年月日
x = datetime.datetime(2019, 7, 26, 13, 24, 55).date()
print(x)

# 返回一个当前具体的时间
x = datetime.datetime(2019, 7, 26, 13, 24, 55).now()
print(x)

x = datetime.datetime.today().hour
print(x)

# 返回星期几
x = datetime.datetime(2019, 7, 26, 13, 24, 55).weekday()
print(x)

# strptime的用法
x = datetime.datetime(2019, 7, 26, 13, 24, 55).strptime("2019-05-24", "%Y-%m-%d")
print(x)

# class: date
print('*'*20)
today = datetime.time

# class: time
print(today)

# calender模块
print('*'*20)

# weekday()方法
weekday = calendar.weekday(2019, 7, 26)
print(weekday)

# class:
# 获取calendar对象
# 设置firstweekday=6
calendar = calendar.Calendar(firstweekday=6)

# 返回 day
all_dates = calendar.itermonthdates(2019, 6)
for i in all_dates:
    print(i)

# 返回 year-month-day
all_ = calendar.itermonthdays(2019, 6)
for i in all_:
    print(i)

# 以元组形式返回(day,weekday) 其中weekday是0-6
all_ = calendar.itermonthdays2(2019, 6)
for i in all_:
    print(i)

# 以元组形式返回(year, month, day)
all_ = calendar.itermonthdays3(2019, 6)
for i in all_:
    print(i)

# 以元组形式返回(year, month, day, weekday)
all_ = calendar.itermonthdays4(2019, 6)
for i in all_:
    print(i)
