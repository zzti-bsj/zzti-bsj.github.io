python_爬取表情包

python抓取表情包

这是一篇可耐的博客，看完能够抓取到了知乎上的表情包，想想是不是有点小激动？

如果读者努力并且兴趣满满的话，看完笔者的这篇文儿，举一反三是不成问题的。

废话少说，我们开始吧。

1. 获取网页源代码

不用想，首先请求打开一个网页url

url_page = urllib.request.urlopen(url)

然后利用函数根据url_page对象获取网页源代码，毋庸置疑，网页源码到手了才能够进行操作。

html_code = url_page.read().decode()

2. 匹配图片的url

设置正则表达式规则匹配图片url的地址,对获取到的网页源码按规则进行匹配，查找到所需的图片的url

imgre = re.compile(r'<figure>.*?src="(.+?\.jpg)"',re.S)

imglist = re.findall(imgre,html) #返回一个匹配到的图片url列表

3. 根据url将图片下载到本地

获取url资源

urllib.request.urlretrieve() #params: url, '保存路径'

使用这个函数，将资源依次下载到本地，需要首先在保存路径下新建一个保存的文件夹。
到这里，不出意外，运行之后，文件夹里已经有可爱的小表情了。

附上源码如下：

Python爬取表情源码

#coding=utf-8

#urllib模块提供了读取Web页面数据的接口
import urllib.request

#re模块主要包含了正则表达式
import re

#定义一个get_html_code()函数
def get_html_code(url):
    #urllib.request.urlopen()请求打开一个URL地址
    url_page = urllib.request.urlopen(url)  

    #read()方法用于读取URL上的数据
    html_code = url_page.read().decode() 

    return html_code

def get_img(html):
    #re.compile() 可以把正则表达式编译成一个正则表达式对象.
    imgre = re.compile(r'<figure>.*?src="(.+?\.jpg)"',re.S)

    #使用re.findall()按指定规则匹配
    imglist = re.findall(imgre,html)      

    #把筛选的图片地址通过for循环遍历并保存到本地
    #urllib.request.urlretrieve()方法,直接将远程数据下载到本地，图片通过x依次递增命名

    x = 0
    for imgurl in imglist:
     urllib.request.urlretrieve(imgurl,'D:\biaoqing\%s.jpg' % x)
     x += 1


html = get_html_code("https://www.zhihu.com/question/46508954")
get_img(html)
