Java的基础语法

java的基础语法

  自我认为C/C++是我接触编程语言时的母语，经过对C语法长时间的使用，在学习Php、Java、python等编程语言的时候，我个人会进行一些相对于C语言的比较，Java的基础语法和C/C++类似，对我来说应该是能够更快的学习Java。

Jave的基础语法

类型

Java的类型包括两种

  1. 基本类型
    基本类型又包括5类：
    （1）字节型：byte（1个字节）
    （2）整型：short（2个字节）、int（4个字节）、long（8个字节）
    （3）浮点型：float（4个字节）、double（8个字节）
    （4）字符型：char（2个字节）
    （5）布尔型：boolean（Java中没有给定boolean的字节）
  2. 类类型

– Note –
  （1）所谓的基本类型，就是在使用时，考虑一下数据用多少内存比较经济，由JVM为用户在内存中配置与管理
  （2）浮点数在写入的时候默认为是double型的，如果使用float，请使用类型转化

常用的格式控制符

符号 说明
%% 格式化输出一个%
%d 格式化输出整数类型的数，除此之外还有byte、Byte、Short、Integer、Long、BigInteger
%f 十进制浮点数格式输出，除此之外还有Float、Double、BigDecimal
%e、%E 科学计数法浮点数格式输出，除此之外还有Float、Double、BigDecimal，%e代表e小写，%E代表E大写
%o 八进制整数类型格式输出，除此之外还有byte、Byte、Short、Integer、Long、BigInteger
%x、%X 十六进制整数格式化输出，除此之外还有byte、Byte、Short、Integer、Long、BigInteger
%s、%S 字符串格式化符号
%c、%C 字符格式输出提供的数必须是byte、short、char、Byte、Short、Charactor或者是Integer，%c表示字母小写，%C表示字母大写
%b、%B 输出boolean值，%b代表小写true/false，%B代表大写TRUE/FALSE
%h、%H 使用Integer.toHexString(arg.hashCode())来得到输出结果，如果arg是null，则输出null，也常用于十六进制格式输出
%n 输出平台特定的换行符号，Windows下是”\r\n”，Linux下是’\n’

变量

  1. 命名规范
    （1）不可为关键字、保留字
    （2）变量名不可数字开头
    （3）不可使用特殊符号
  2. 命名习惯
      Java的命名习惯，通常以小写字母开头，并且在每个单词的开头大写，这种命名方式称为驼峰命名法
      如：int ageOfStudent;

常量

  1. 几种常见进制数的表示方法
    （1）二进制数（0b开头）：int num = 0b10010011;
    （2）八进制数（0开头）：int num = 014;
    （3）十进制数：int num = 12
    （4）十六进制数（0x开头）：0xC
  2. 常见的转义字符
转义字符 说明
\\ 反斜杠\
\‘ 单引号’
\“ 双引号”
\uxxxx 以十六进制指定Unicode字符输出，x表示数字
\xxx 以八进制指定Unicode字符输出，x表示数字
\b 倒退一个字符
\f 换页
\n 换行
\r 光标移至行首
3. Java中的常量表示法
  从Java SE7之后，撰写整数或者浮点数能够使用下划线更清楚的表示某些数字，如：
  int number = 1234_5678;
  int mask = 0b1010_1010_1010;

运算符

  1. 算数运算符
    + - * / %
  2. 比较、条件运算符
    > < >= <= == !=

– Note –
  三目运算符：?: –> 例：a>b?a:b;
3. 逻辑运算符
(与)&& (或)|| (非)！
4. 位运算符
(移位运算符)>> << (逻辑与)& (逻辑或)| (异或)^ (非)~
5. 递增递减运算
++ --
6. 指定运算符
+= -= /= %= ^= &= 等等

类型转换的经度问题

  前面在基本类型介绍的时候已经举例说明float的转换方法了，在这里简单说几点要点。
  1、运算时
  （1）如果表达式中包括不同类型数值，则运算时以长度最长的类型为主，其他数值自动提升类型。
  （2）Java在运算整数时，如果全部的操作数都不是大于int的，一律在int的空间中运算。

– Note –
  注意精度遗失的发生。
